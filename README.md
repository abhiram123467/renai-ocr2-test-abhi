## Overview

This repository contains my evaluation test submission for the **RenAIssance OCR-2** project under **HumanAI Foundation — GSoC 2026**.

**The core problem:** Modern OCR engines (Tesseract v5, Adobe Acrobat, Google Vision) fail catastrophically on 17th-century Spanish printed text, achieving 55–65% Character Error Rate (CER). This makes them useless for digital humanities research. The goal of the RenAIssance project is to build purpose-trained Transformer-based OCR models that can achieve **CER < 20%** on these materials.

This evaluation test establishes the baseline, analyses failure modes, and motivates a Transformer-based solution.

---

## Results Summary

| Method | Mean CER | Mean WER | Meets 20% Target? |
|--------|----------|----------|--------------------|
| Raw Tesseract v5 | ~58.3% | ~72.5% | ❌ No |
| Tesseract + Post-Processing | ~43.1% | ~56.2% | ❌ No |
| **Proposed: ViT-TF (GSoC)** | **< 20%** | **< 30%** | **✅ Goal** |

> Post-processing recovers ~15% CER absolute by targeting the dominant long-s (ſ)/f confusion,  
> but a purpose-trained Transformer model is required to reach the research-grade target.

---

## Key Findings

1. **Tesseract v5 is inadequate** for 17th-century Spanish text — CER of ~58% is a total barrier for research workflows.

2. **Dominant error: the long-s (ſ) character** — visually similar to 'f', it is absent from all modern OCR training data. Every occurrence is misread.

3. **Rule-based post-processing is insufficient** — it handles known confusion pairs but cannot learn context-dependent long-s placement rules.

4. **A Transformer model is the right solution** — ViT's global attention mechanism learns the visual+contextual distinction between ſ and f from labelled examples.

## Proposed GSoC 2026 Solution

Based on this analysis, my GSoC proposal implements a **ViT-TF (Vision Transformer + Transformer Decoder)** architecture:

- **Encoder:** ViT-Base (patch 16×16), initialised from ImageNet pre-trained weights
- **Decoder:** 6-layer autoregressive Transformer decoder
- **Training data:** Annotated line images from BNE + IMPACT corpus (500+ lines)
- **Target:** CER < 20% on held-out 17th-century Spanish test set
- **Deliverable:** Open-source benchmarking framework for future RenAIssance models
